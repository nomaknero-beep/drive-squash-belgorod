# Сайт сквош-клуба «Драйв» (Белгород)

Домен: **drive-squash.ru**

Статический фронтенд (HTML + CSS + ванильный JS) + PHP-бэкенд (заявки в
Telegram) + MySQL (рейтинг и турниры). Без сборщиков и Composer-зависимостей —
заводится на любом хостинге/сервере с PHP 7.4+ и MySQL 5.7+.

## Структура
- `index.html`, `css/`, `js/` — фронтенд (одностраничник).
- `images/` — постеры, логотип, галерея (13 фото), видео.
- `api/` — PHP-бэкенд: бронь → Telegram (`booking.php`), рейтинг (`rating.php`).
- `db/schema.sql` — таблицы рейтинга/турниров + тестовые данные.
- `api/config.example.php` — шаблон секретов. Реальный `api/config.php` в git НЕ
  лежит (в `.gitignore`) — создаётся на сервере.

## Запуск на сервере (PHP + MySQL)
1. Клонировать репозиторий в корень сайта:
   `git clone https://github.com/nomaknero-beep/drive-squash-belgorod.git`
2. Поднять веб-сервер (Apache/nginx) с PHP 7.4+ на этой папке.
3. Скопировать `api/config.example.php` → `api/config.php` и заполнить:
   `TG_TOKEN`, `TG_CHAT_ID`, `DB_HOST`, `DB_NAME`, `DB_USER`, `DB_PASS`.
4. Создать БД MySQL, импортировать `db/schema.sql`.
5. Убедиться, что `api/.htaccess` действует (закрывает `config.php` и логи от
   доступа по HTTP). На nginx — настроить запрет на `*.php` в `api/` для конфига
   и `*.log` вручную.
6. Направить домен **drive-squash.ru** на сервер (DNS A-запись на IP сервера).

## Настройка Telegram-бота (для заявок брони)
1. В Telegram написать @BotFather → `/newbot`, задать имя → получить **токен**.
2. Написать своему боту любое сообщение (чтобы появился чат).
3. Узнать **chat_id**: открыть `https://api.telegram.org/bot<ТОКЕН>/getUpdates`
   и найти `"chat":{"id":...}`.
4. Вписать токен и chat_id в `api/config.php`.

## Наполнение данными
- Цены / тренеры / преимущества — `js/data.js`.
- Галерея — файлы `images/gallery/*.jpg`, список в `js/data.js` (массив `gallery`).
- Рейтинг / турниры — таблицы `players`, `tournaments` в MySQL (правка через
  phpMyAdmin или SQL).

## Заметки
- Все тексты цен/тренеров сейчас — плейсхолдеры, заменить реальными данными клуба.
- Видео в секции рейтинга встроено через VK-плеер (iframe). Локальная копия
  клипа лежит в `images/` — при желании можно переключить на самостоятельную
  отдачу через `<video>`.
